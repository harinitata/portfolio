Projects

LunaCare - a girl health chatbot

link to github repo

LunaCare is a web-based platform designed to empower women by providing accessible and supportive health information, focusing on an intelligent chatbot. Addressing the challenges women face in finding reliable and empathetic resources, particularly concerning menstruation, PCOD, PCOS, Breast health, LunaCare offers a safe space for users to get their questions answered and receive helpful guidance. The chatbot leverages natural language processing to deliver a friendly, informative, and supportive experience.

PointRisk - forest fire prediction system

link to github repo

Built a web-based Forest Fire Prediction System that forecasts wildfire risk using real-time weather data. The system integrates a Flask backend with a machine learning model (Random Forest Classifier) trained on environmental data like temperature, humidity, and wind speed. A user-friendly frontend built with HTML and Tailwind CSS allows users to input city names and view wildfire risk along with current weather conditions in a visually appealing dashboard.
